We will train a CNN at Browser with 2 hidden layers

  - **Layer Convolution**
    - input **[28, 28, 1]**
    - kernel size: **5**
    - activation Function: **reLu**
    - number of Filters: **8**
    - strides: **1**
  - **Layer MaxPooling**
      - pool Size: **2**
      - strides: **2**
  - **Last Layer**
      - Softmax Layer with **10 classes**
